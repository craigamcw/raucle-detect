# Raucle proposals

Forward-looking design documents for the raucle project. Not yet implemented; intended as artefacts to share, gather feedback on, and use as the basis for follow-up papers and protocol-extension submissions.

| Proposal | Status | Target |
|---|---|---|
| [VCD-A2A: Capability Discipline for the Agent-to-Agent Protocol](vcd-a2a.md) | Draft 2026-05-18 | A2A protocol extension submission |
| [TEE-anchored VCD gate](tee-gate.md) | Draft 2026-05-18 | Q3 2026 commercial-product feature; paper §8 update for camera-ready |

Each proposal:
- States the threat-model addition (if any) over the base VCD primitives in the IEEE S&P 2027 paper draft.
- Specifies the on-wire / on-disk format extension.
- Lists backwards-compatibility constraints.
- Names a reference-implementation milestone schedule.

Feedback: open an issue on the repository or email `commercial@raucle.com` for cross-organisation collaboration discussions.
